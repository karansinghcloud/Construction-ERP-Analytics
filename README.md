# 📌 Project Overview

This project simulates a real-world Construction ERP system used to monitor projects, procurement, inventory, vendors, budgets, manpower, and site execution.

The objective is to centralize operational data and transform it into interactive dashboards that support faster and data-driven business decisions.

---

# 🎯 Business Problem

Construction companies manage multiple sites, vendors, materials, and budgets simultaneously.

Without centralized reporting, businesses face:

- Material shortages
- Vendor delays
- Budget overruns
- Inventory mismatch
- Site execution delays
- Resource allocation issues

This ERP analytics solution helps management monitor the entire project lifecycle.
